.. _library_hierarchies:

``hierarchies``
===============

This library provides objects implementing reflection predicates over
class and prototype hierarchies.

Документация интерфейса прикладного программирования
-----------------

Open the
`../../docs/library_index.html#hierarchies <../../docs/library_index.html#hierarchies>`__
link in a web browser.

Загрузка
-------

Чтобы загрузить все сущности этой библиотеки, используйте файл ``loader.lgt ``:

::

   | ?- logtalk_load(hierarchies(loader)).

Тестирование
-------

Чтобы проверить предикаты этой библиотеки, используйте файл ``tester.lgt ``:

::

   | ?- logtalk_load(hierarchies(tester)).
